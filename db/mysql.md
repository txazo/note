### 查询缓存

查看查询缓存状态:

```
show variables like '%query_cache%';
```

查询缓存开启后:

* `select`语句会先经过查询缓存检查是否命中
* 表结构或数据改变，和这个表相关联的所有缓存都会失效
* 避免频繁的写操作，可使用批量写
